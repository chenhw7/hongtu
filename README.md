# 鸿图建材获客管理工具

江门鸿图建材有限公司内部的客户获客管理工具，帮助业务团队自动采集政府采购招标线索、管理客户信息和跟进记录。

---

## 一、系统简介

本工具是一个网页版应用，包含四大功能模块：

| 模块 | 功能说明 |
|------|----------|
| **仪表盘** | 首页，展示客户总数、今日新增、待跟进数量等统计信息 |
| **客户管理** | 管理客户信息，包括新增、编辑、删除、跟进记录、状态管理、Excel导入导出 |
| **线索管理** | 查看爬虫采集的招标线索，支持筛选、搜索、一键转客户、批量转化、导出Excel |
| **爬虫控制** | 手动触发数据采集，查看采集任务历史和运行状态 |

## 二、安装步骤

### 1. 安装 Python

1. 访问 [Python 官网](https://www.python.org/downloads/) 下载 Python 3.10 或以上版本
2. 运行安装程序，**务必勾选 "Add Python to PATH"**（添加到环境变量）
3. 安装完成后，打开命令提示符（按 `Win+R`，输入 `cmd`），输入 `python --version` 确认安装成功

### 2. 安装 uv（Python 包管理器）

本项目使用 [uv](https://docs.astral.sh/uv/) 管理依赖和虚拟环境，安装方法：
1. 打开命令提示符（`Win+R` 输入 `cmd`）
2. 执行命令：`pip install uv`
3. 输入 `uv --version` 确认安装成功

> 提示：`start.bat` 启动脚本会自动检测 uv，若未安装会尝试自动安装，因此也可跳过此步直接使用启动脚本。

### 3. 启动系统

1. 双击文件夹中的 **`start.bat`** 文件
2. 首次启动会自动创建虚拟环境并安装所需依赖（需要联网），请耐心等待
3. 看到"服务已启动"提示后，打开浏览器访问 **http://localhost:5000**

> 之后每次使用，只需双击 `start.bat` 即可。脚本会自动处理虚拟环境和依赖。

<details>
<summary>手动启动（不使用 start.bat）</summary>

```bash
# 创建虚拟环境
uv venv

# 安装依赖
uv pip install -r requirements.txt

# 启动服务
uv run python run.py
```

</details>

## 三、使用说明

### 仪表盘（首页）

打开浏览器访问系统后，首先看到的是仪表盘页面，展示：
- **总客户数** — 点击可跳转到客户列表
- **今日新增** — 今天新添加的客户数量
- **待跟进客户** — 有到期跟进计划的客户数量
- **最近跟进记录** — 最近5条跟进记录
- **最近新增客户** — 最近5条新增客户

### 客户管理

点击左侧菜单"客户管理"进入，可以：

- **新增客户** — 点击"新增客户"按钮，填写公司名称（必填）、联系人、电话等信息
- **编辑客户** — 点击客户列表中的"编辑"按钮修改信息
- **查看详情** — 点击客户名称或"详情"按钮，查看完整信息和跟进记录
- **添加跟进** — 在客户详情页底部，填写跟进内容、方式、下一步计划和跟进日期
- **快速改状态** — 在客户详情页右上角下拉框中直接切换客户状态
- **导入Excel** — 点击"导入Excel"，上传 .xlsx 文件批量导入客户
- **导出Excel** — 点击"导出Excel"下载所有客户数据

客户状态说明：新线索 → 跟进中 → 已报价 → 已成交 / 已流失

### 线索管理

点击左侧菜单"线索管理"进入，可以：

- **筛选线索** — 按来源类型、发布日期、转化状态、关键词筛选
- **查看详情** — 点击项目名称查看线索完整信息
- **一键转客户** — 点击"转客户"按钮，将线索自动创建为客户记录
- **批量转客户** — 勾选多条线索后点击"批量转客户"
- **导出Excel** — 点击"导出Excel"下载当前筛选的线索数据
- **查看原文** — 点击外链按钮跳转到政府采购网原文页面

### 爬虫控制

点击左侧菜单"爬虫控制"进入，可以：

- **快速采集** — 点击数据源卡片上的"立即采集"按钮
- **配置采集** — 选择数据源、设置页数和关键词后点击"开始采集"
- **查看任务历史** — 页面下方表格显示所有采集任务的执行记录
- **自动采集** — 系统每天早上 8:00 自动执行采集任务（需保持程序运行）

采集的数据源：
- **中国政府采购网** (ccgp) — 全国范围的政府采购信息
- **广东省政府采购网** (gdgpo) — 广东省内采购信息

## 四、局域网访问方法

本工具支持局域网内其他电脑访问，方法如下：

### 1. 查看本机IP

在启动程序的电脑上，打开命令提示符（`Win+R` 输入 `cmd`），输入：
```
ipconfig
```
找到"IPv4 地址"，类似 `192.168.1.100`。

### 2. 局域网访问

在其他电脑的浏览器中输入：
```
http://192.168.1.100:5000
```
（将 `192.168.1.100` 替换为实际IP地址）

### 3. 防火墙设置

首次局域网访问时，Windows 可能弹出防火墙提示，请选择"允许访问"。如果未弹出提示且无法访问：

1. 打开"控制面板" → "Windows Defender 防火墙" → "允许应用通过防火墙"
2. 找到 Python，勾选"专用"和"公用"网络
3. 如果列表中没有 Python，点击"允许其他应用"手动添加

## 五、数据备份方法

### 方法一：使用备份脚本

1. 双击文件夹中的 **`backup.bat`** 文件
2. 备份文件会自动保存到 `backups` 文件夹中，文件名包含日期时间
3. 建议每周至少备份一次，重要时期每天备份

### 方法二：手动复制

直接复制 `instance` 文件夹下的 `hongtu.db` 文件到U盘或其他安全位置。

### 备份建议

- 定期将 `backups` 文件夹复制到U盘或网盘
- 重大操作前（如批量导入、系统更新）先做一次备份
- 保留最近 2-3 个月的备份文件

## 六、常见问题解答

### Q：双击 start.bat 提示"未检测到 Python"？

**A：** Python 未安装或未添加到环境变量。请重新安装 Python，安装时勾选 "Add Python to PATH"。安装后重新打开命令提示符输入 `python --version` 确认。

### Q：启动后浏览器打开是空白页？

**A：** 请确认浏览器地址栏输入的是 `http://localhost:5000`（注意是 http 不是 https）。如果页面一直加载，请等待几秒让程序完全启动。

### Q：其他电脑无法通过局域网访问？

**A：** 请检查以下几点：
1. 两台电脑是否在同一个局域网（连同一个WiFi/路由器）
2. 启动程序的电脑上 Windows 防火墙是否允许 Python 通过
3. IP地址是否正确（在启动程序的电脑上用 ipconfig 查看）
4. 确保程序正在运行（黑窗口不要关闭）

### Q：启动时提示"端口被占用"？

**A：** 说明已有另一个程序在使用 5000 端口。解决方法：
1. 关闭之前启动的程序窗口
2. 打开任务管理器，结束所有 python.exe 进程
3. 重新双击 start.bat

### Q：爬虫采集没有数据？

**A：** 政府采购网站可能有反爬机制，采集结果取决于网站结构和网络状况。可以尝试：
1. 减少每次采集的页数（设置为1-2页）
2. 等待几分钟后重试
3. 检查网络连接是否正常

### Q：数据会丢失吗？

**A：** 数据保存在本地数据库文件中，正常使用不会丢失。但建议定期使用 `backup.bat` 备份数据，以防电脑故障。

### Q：如何停止服务？

**A：** 在运行程序的黑色命令行窗口中按 `Ctrl + C` 即可停止服务。直接关闭窗口也可以。

### Q：可以同时在两台电脑上运行吗？

**A：** 不建议。同时运行两个实例可能导致数据库冲突。建议在一台电脑上运行，其他电脑通过局域网访问。

## 三、已放弃数据源（4 个）

| 数据源 | 网站 | 放弃原因 |
|--------|------|----------|
| ctbpsp.com | 中国招标投标公共服务平台 | **5 层反爬**：阿里云 WAF + Vaptcha 验证码 + 网易易盾 + 反调试 + DES 加密，代码已删除 |
| h2o-china.com | 中国水网 | **数据失效**：招标信息栏目已下线/停止更新 |
| ndrc.gov.cn | 国家发改委 | **技术障碍**：需登录 + Dojo 框架加密 + 接口返回空数据 |
| ggzyjy.shandong.gov.cn | 山东省公共资源交易 | **不可用**：旧平台 502，新系统需登录认证 |

> 详细分析见 `docs/ctbpsp-采集可行性调研报告.md`、`docs/发改委-采集可行性调研报告.md`。

## 四、不推荐实现（4 个）

| 数据源 | 不推荐原因 | 替代方案 |
|--------|-----------|---------|
| 中国采购与招标网 | 数据与 ccgp + ggzyjy 高度重叠；需付费 500-2000 元/月 | 现有数据源已覆盖同类信息 |
| 企业画像（爱企查/天眼查/企查查） | 爱企查无官方 API 且 TLS 指纹检测；天眼查类似 | 企查查开放平台（openapi.qcc.com）按次付费可考虑，需企业认证 |
| B2B 平台（1688/慧聪） | 1688 反爬极高，慧聪网 IP 封禁，Playwright 难持久 | 人工操作：1688 商家搜索筛选后手动录入系统 |
| 其他方向性规划 | 招标雷达/建设通/四库一平台等仅有方向性提及，无技术可行性验证 | 待后续需要时再调研 |

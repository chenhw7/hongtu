## 二、已完成工作

### POI 全国扩展 ✅

**实现概要**：
- `scraper/poi_regions.py`：全国 30 个省级行政区约 100 个重点城市配置
- 支持分批采集（`get_batch_cities()`），控制高德 API 每日配额（5000 次/天）
- `POI_SCOPE` 配置项：`'guangdong'`（默认，仅广东 21 城）/ `'national'`（全国）
- 向后兼容：现有广东省采集流程不受影响

**城市选取策略**：
- 每省选取省会 + 经济强市（GDP 前 3）
- 广东省保留完整 21 城市覆盖
- 全国总计约 100 个重点城市

**技术细节**：
- 城市配置格式：`(省名, 城市名, 城市编码)` 元组列表
- `get_batch_cities(batch_index, batch_size)` 支持按批次返回城市子集
- 每次采集间隔 1-2 秒随机延迟，避免触发高德频率限制

**测试覆盖**：26 个单元测试，覆盖城市配置、分批逻辑、边界场景等

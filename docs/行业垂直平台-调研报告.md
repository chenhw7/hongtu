# 行业垂直平台采集可行性调研报告

> 调研日期：2026-07-16
> 调研范围：中国管道商务网、中国水网、北极星环保网、中国采购与招标网
> 结论：**无直接可用数据源**，部分平台可通过 Playwright 突破，但投入产出比不高

---

## 一、调研背景

当前系统已覆盖 ccgp（全国政府采购）、gdgpo（广东省政府采购）、eia（广东环评）、ggzyjy（广东公共资源交易）、poi（高德 POI）五大数据源。为拓展管道行业垂直领域信息，对四个行业平台进行了可行性调研。

---

## 二、平台逐一调研

### 1. 中国管道商务网（chinapipe.net）

**平台概况**：管道行业专属门户网站，包含招标信息、供求发布、行业资讯。

**调研结论**：⚠️ 部分可行（需 Playwright）

| 维度 | 评估 |
|------|------|
| 数据价值 | 高 — 管道行业专属，招标/供应信息精准匹配 |
| 反爬防护 | 中 — 使用 JS 挑战（Cloudflare 或自建），需浏览器渲染 |
| API 可用性 | 无公开 API，页面需 JS 渲染后获取 |
| 技术难度 | 中 — Playwright 首次加载获取 Cookie 后可 httpx 请求 |
| 数据量 | 中等，日均更新 5-20 条 |

**技术方案**（如实施）：
- `scraper/chinapipe/` 模块
- Playwright 首次加载突破 JS 挑战
- 解析渲染后的 HTML 提取招标列表
- 详情页可能需二次 Playwright 请求

**建议**：P1 优先级，待 Playwright 基础设施就绪后实施。

---

### 2. 中国水网（h2o-china.com）

**平台概况**：水务行业权威媒体，覆盖供水、排水、污水处理等领域。

**调研结论**：❌ 不可行（招标栏目已下线）

| 维度 | 评估 |
|------|------|
| 数据价值 | 原本高 — 水务项目管道用量大 |
| 当前状态 | 招标信息栏目已下线/停止更新 |
| 可用数据 | 仅存新闻资讯，无结构化招标数据 |
| 替代方案 | 水务项目已可通过 ccgp/ggzyjy 关键词覆盖 |

**建议**：放弃。水务/污水处理项目已通过现有 ccgp（政府采购）和 ggzyjy（工程建设）数据源覆盖，关键词"污水处理厂""供水管网"等已在 `scraper/keywords.py` 中。

---

### 3. 北极星环保网（bjx.com.cn）

**平台概况**：环保行业综合门户，覆盖污水处理、固废、大气、土壤等领域。

**调研结论**：⚠️ 部分可行（WAF 可绕过）

| 维度 | 评估 |
|------|------|
| 数据价值 | 中-高 — 污水处理/固废项目，管道需求明确 |
| 反爬防护 | 中 — WAF（Web Application Firewall），但可通过请求头伪装绕过 |
| API 可用性 | 列表页为服务端渲染 HTML，可直接解析 |
| 技术难度 | 中 — 需设置完整浏览器请求头（UA/Referer/Cookie） |
| 数据量 | 较大，日均更新 10-30 条 |

**技术方案**（如实施）：
- `scraper/bjx/` 模块
- httpx + 浏览器请求头伪装
- WAF 绕过：设置合理的 Cookie 和 Referer
- 如遇拦截，降级为 Playwright

**建议**：P2 优先级，可在 Playwright 基础设施就绪后与中国管道商务网一起实施。

---

### 4. 中国采购与招标网（chinabidding.com.cn）

**平台概况**：老牌招标信息聚合平台，汇聚全国各类招标信息。

**调研结论**：⚠️ 需付费 API

| 维度 | 评估 |
|------|------|
| 数据价值 | 中 — 信息聚合平台，数据源与 ccgp/ggzyjy 有重叠 |
| 免费访问 | 仅可查看摘要，完整信息需登录+付费 |
| API 可用性 | 提供付费数据 API，按条/包月计费 |
| 技术难度 | 低（如有 API）/ 高（如需爬取） |
| 成本 | 付费 API 约 500-2000 元/月，视数据量而定 |

**建议**：P3 优先级。由于数据源与现有 ccgp/ggzyjy 有大量重叠，增量价值有限。如确需接入，建议先评估付费 API 的性价比。

---

## 三、调研汇总

| 平台 | 代号 | 可行性 | 优先级 | 关键技术 | 备注 |
|------|------|--------|--------|---------|------|
| 中国管道商务网 | chinapipe | ⚠️ 部分可行 | P1 | Playwright | 管道行业专属，价值最高 |
| 中国水网 | h2o | ❌ 不可行 | — | — | 招标栏目已下线 |
| 北极星环保网 | bjx | ⚠️ 部分可行 | P2 | WAF 绕过 | 环保项目覆盖 |
| 中国采购与招标网 | chinabidding | ⚠️ 需付费 | P3 | 付费 API | 与现有源重叠 |

## 四、总体建议

1. **短期**：不新增行业垂直平台，现有数据源 + 50 个分类关键词已能覆盖大部分管道相关项目
2. **中期**：待 Playwright 基础设施就绪后，优先接入中国管道商务网（P1）
3. **长期**：根据实际需求评估北极星环保网和付费 API 的必要性
